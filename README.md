This repository contains my daily practice of **Data Structures and Algorithms (DSA)** using **C++**.

I'm using this repo to track consistent, day-to-day problem solving and to build a strong foundation in DSA for interviews and competitive programming.

## 📖 About

Consistency beats intensity. Instead of cramming, this repo is my public log of solving at least one DSA problem a day — building intuition, speed, and confidence over time.

## 📂 Structure

Each file is named by the date it was solved (e.g. `6-7-2026.txt`), containing the problems and solutions worked on that day.

```
cpp-dsa-practice/
├── 1-6-2026.txt
├── 2-6-2026.txt
├── 3-6-2026.txt
├── ...
└── README.md
```

## 📚 Topics Covered

- Arrays
- Strings
- Linked Lists
- Stacks & Queues
- Recursion & Backtracking
- Trees (Binary Trees, BST)
- Graphs (BFS/DFS, Shortest Path)
- Sorting & Searching
- Dynamic Programming
- Greedy Algorithms
- Hashing
- More topics added as I progress

## 🚀 How to Run

Each solution can be compiled and run individually using any standard C++ compiler:

```bash
g++ filename.cpp -o output
./output
```
